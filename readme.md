Design Pattern thường chia làm 3 nhóm:
    1. Creational – Cách tạo object
    2. Structural – Cách ghép object/class với nhau
    3. Behavioral – Cách object giao tiếp (chưa bàn ở đây)

CREATIONAL DESIGN PATTERN: Tạo object như thế nào cho linh hoạt, dễ mở rộng, dễ test
STRUCTURAL DESIGN PATTERN: Làm sao ghép các object/class lại với nhau mà không phụ thuộc chặt
